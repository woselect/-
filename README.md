# 可爱的小音乐播放器

实现了音乐播放器的一些基本功能，可以播放本地、网络的音乐，切换播放顺序类型，收藏音乐，搜索网络音乐，下载网络音乐等等功能。关于网络音乐的获取使用了百度音乐的api.
